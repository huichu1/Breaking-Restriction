# REBUTTAL EXPERIMENT
This folder mainly contains 4 experiments result for our rebuttal.

## The content in this folder:
This folder mainly contains results below:

**depth_test/** : directory with code of testing our method on a network with large depth. (30-5-6-7-8-9-10-11-12-10)

**width_test/** : directory with code of testing our method on a network with large width. (100-110-120-128-10)

**time_and_query_test/** : directory with code of testing time and query consumption in our attack process.

**activation_test/** : directory with code of testing the activatiing atatus for every neurons in PReLU FCN, related to **"Is the Hard-Label Cryptanalytic Model Extraction Really Polynomial?"**.
 
Readers can check the experiment results and code in each directory, or use this code to test the original models in form repository after editing properly. Notice that we do not upload all dual points in experiment due to the restriction of github
